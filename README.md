# Sign Translation Glove Website

This repository contains the website for the Sign Translation Glove project.

## Overview

The site presents a wearable translation prototype designed to convert sign language gestures into readable text. It includes:

- the project mission and accessibility focus
- prototype workflow and build process
- electronics, 3D design, and software sections
- market context and experimental results

## Tech

- HTML
- CSS
- JavaScript

## Local preview

Open [index.html](./index.html) directly in a browser, or run a simple local server:

```bash
python -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.
